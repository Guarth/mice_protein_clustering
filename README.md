# mice-protein-clustering
Unsupervised learning on biomedical protein data using PCA, KMeans, and GMM
Overview

This project applies unsupervised learning techniques to cluster mouse brain protein expression data and evaluate alignment with biological conditions.

⚙️ Techniques Used
K-Means Clustering
Gaussian Mixture Models (EM)
PCA & Kernel PCA
Silhouette Score, ARI, NMI
Pseudo-label classification using Nearest Class Mean
Hungarian Algorithm for label alignment
